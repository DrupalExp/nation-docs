# 5.7 Button shortcode

Create a button element.

### Attributes 

**Title**: button title

**Icon**: icon along with title button

**Button Type**: type of button. It can be:

* Default 
* Primary 
* Success 
* Info 
* Warning 
* Danger 
* Link

 **Button Size**: set size for button. It can be:

* Large 
* Small 
* Extra 
* Small

 **Class**: custom class for button.
